动态规划逻辑前提：
1.最优子结构
我们可以通过子问题的最优解推导出问题的最优解，后面阶段的状态可以通过前面的状态推导出来
2.无后效性
1）推导后边阶段时我们只关心前面阶段的状态值，不关心怎么推导出来的2）某阶段状态一旦确定就不受之后阶段的决策影响。
3.重复子问题
不同决策序列，到达某个相同的阶段时，会产生重复的状态。
